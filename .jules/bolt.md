## 2024-05-24 - Algorithmic Loop Benchmark
**Learning:** When benchmarking algorithmic loops, using MagicMock objects inside the loop (e.g., mocking BeautifulSoup tags) can dominate execution time and mask actual algorithmic performance gains.
**Action:** Use simple native Python types (like strings) in benchmarks to measure true speedup.
